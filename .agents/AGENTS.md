# Regras do Projeto (smartnotes-ai)

* **Instruções do GitHub:** Sempre que fizer alterações de código no projeto, inclua no final da resposta as instruções detalhadas e passo a passo de como o usuário pode enviar essas alterações para o repositório do GitHub (usando `git status`, `git add`, `git commit` e `git push`).
* **Idioma dos Commits:** Todos as mensagens de commit (`git commit -m "..."`) fornecidas nas instruções devem ser escritas obrigatoriamente em **inglês** (ex: `git commit -m "fix: paste text as plain text without styles"`).
